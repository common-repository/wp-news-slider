== Description ==

Show a widget with news that you add in your blog.

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.
4. After yo activate the plug in, you will see an new option called "WP News Slider".

== Adding message ==
5. Go to the "WP News Slider" option in the Settings menu.
6. Click in the "Add New Message"

== Configuring the Widget ==
   
7. Once you have added messages, go to the "Widgets" section in the "Design" menu. 
8. Look for the "WordPress News Slider" widget an clik in the "add" link.
9. When the widget was added to the click on the "edit" link.
10. There, you can configure the time thats one message will stay in the screen
    before another message appears, the date format and the link's text.
11. Press "Save Changes". Done !
